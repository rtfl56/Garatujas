# Resumo pra os colegas


Esta disciplina tem o objetivo de te passar os conhecimentos basicos para fazer paginas web, como hml, css, e javascript, com a metodologia passada na aula sendo de facil entedimento e acessivel para iniciantes na area.

## Explicação HTML
html significa linguagem de marcação de hipertexto, que ja deixa explicito que não é uma linguagem de progamação, pois não se usa lógica de progamação nela, sendo organizada por varios elementos que definem o tipo dos conteudos e oque cada coisa faz na pagina por tags, como <head> que são aonde fica os dados nao aparentes na pagina, como o titulo da pagina e o conjunto de caracteres(charset), e o <body> é todos os dados que fazem parte da página, como os títulos e subtítulo(h1, h2, h3 e etc).

### Explicação CSS
CSS significa Folhas de Estilo em Cascata, é uma linguagem de estilo para modificar toda a aparência visual de página web sem mexer na estrutura do html. O CSS controla o grid, as cores,tamanho de containers e pode estilizar múltiplas páginas ao mesmo tempo.
